# Core Data Structures

This directory contains header files for the core data structures used in the solver.

## Components
- Energy grid (N_E=256, log-spaced, bin-edge based)
- Angular grid (N_theta=512)
- LOCAL_BINS encoding/decoding
- Block ID encoding (24-bit)
- PsiC sparse buffer structure
- OutflowBucket structure
